# Προσωπικό Βιογραφικό - Άγγελος Αραπιώτης

Αυτό το project είναι μια απλή στατική ιστοσελίδα τύπου βιογραφικού, υλοποιημένη με HTML και CSS. Περιλαμβάνει βασικές πληροφορίες για εμένα, τις δεξιότητές μου, και στοιχεία επικοινωνίας.

## 📁 Περιεχόμενα
- `index.html`: Κύρια ιστοσελίδα
- `style.css`: Φύλλο στυλ
- `img/profile.jpg`: Εικόνα προφίλ
- `README.md`: Περιγραφή project

## 🚀 Τεχνολογίες
- HTML5
- CSS3
- (προαιρετικά) Tailwind CSS

## 🎓 Σκοπός
Εκπαιδευτική εργασία για παρουσίαση προσωπικού βιογραφικού με δυνατότητα φιλοξενίας στο Netlify/Vercel μέσω GitHub.
